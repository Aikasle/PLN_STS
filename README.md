# Semantic textual similarity (STS)

Esto es una rapida descripcion de los ficheros que vamos a encontrar en el src.
 * z0\_data.py: visualizar el dataset
 * z1\_embeddings.py: evaluar dataset en sentence embeddings (all-mini)
 * z1\_ngrams.py: evaluar dataset en 1-grama
 * z2\_embeddings\_finetuning.py: fine tunning del sentence embeddings en EN
 * z3\_multiidioma.py: evaluar ES sobre modelo fine tuneado
 * z4\_es.py: fine tunning en ES sobre el modelo all-mini
 * z4\_zh.py: fine tunning en CH sobre el modelo all-mini
 * z4\_multi.py: evaluar el modelo multilingue con ch, es y en
